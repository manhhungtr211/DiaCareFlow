# BR-002: DiaCareFlow — Hỗ trợ bệnh tiểu đường 
## Goal (Mục tiêu Tuần 1)
Xây dựng được một Agent gồm: Harm Assessment Agent, Supervior Agent và Response Agent và 1 RAG Agent (kết quả RAG được đưa vào Response agent). Sử dụng fastAPi để làm giao diện tạm thời để test RAG và các Agent -> Nối với giao diện UI đơn giản để test luồng logic 

## Success Metrics (Chỉ số đo lường)
- Sử dụng Flow trên giao diện Reactjs
- RAG truy xuất nội dung đạt trên 90% 
- Chặn được 100% các câu hỏi vi phạm an toàn y tế
## In Scope 
- Các node Langgraph core if/else -> mở rộng lên các agent
- Chạy được luồng logic các node langgraph sử dụng trên UI
- Static UI: Chỉnh sửa giao diện tĩnh (reactjs) sang branding y tế

## Out of Scope
- Các Agent nâng cao
- Web Search
- Document Pipeline Động
- Hệ thống User & History
- API End-to-End & Streaming
- Deployment
